# Project Charter – PhishGuard

## Project Name
PhishGuard

## Team Name
PhishGuard Team

---

## Project Overview
PhishGuard is a cybersecurity-focused web application concept designed to help users identify suspicious emails, text messages, and URLs by evaluating phishing indicators and providing explainable risk analysis.

The platform aims to improve phishing awareness and help users make safer cybersecurity decisions before interacting with suspicious communications.

---

## Project Purpose
The purpose of this project is to design and manage a realistic cybersecurity product using Agile/Scrum methodologies while producing strong project documentation, cybersecurity analysis, and design artifacts.

Capstone 1 emphasizes project management, teamwork, requirements engineering, and cybersecurity planning rather than full production implementation.

---

## Problem Statement
Phishing remains one of the most common cybersecurity threats affecting individuals and organizations.

Many users lack the technical knowledge required to confidently identify suspicious communications before interacting with them, increasing the likelihood of credential theft, malware infection, financial fraud, and data compromise.

PhishGuard addresses this problem by providing an educational phishing risk analysis platform.

---

## Project Objectives
The project aims to:
