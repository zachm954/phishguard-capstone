# Project Charter – PhishGuard

## Project Name

PhishGuard

## Team Name

PhishGuard Team

---

## Project Overview

PhishGuard is a cybersecurity-focused web application prototype designed to help users recognize suspicious emails, text messages, and URLs by evaluating common phishing indicators and providing understandable risk explanations and cybersecurity-awareness guidance.

The platform is intended to support safer decision-making before users interact with suspicious digital communications.

---

## Project Purpose

The purpose of PhishGuard is to design, document, and evaluate a realistic cybersecurity product using Agile and Scrum methodologies.

Capstone 1 focuses on project planning, teamwork, requirements engineering, cybersecurity research, user-interface design, prototype development, quality assurance, and preparation for continued implementation during Capstone 2.

---

## Problem Statement

Phishing remains one of the most common cybersecurity threats affecting individuals and organizations.

Many users lack the technical knowledge needed to confidently recognize suspicious communications before interacting with them. This can increase the risk of credential theft, malware infection, financial fraud, and data compromise.

PhishGuard addresses this problem by providing explainable phishing-risk guidance, educational content, and recommended user actions.
